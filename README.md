# BG3 Mod Tracker

The goal of this project is to create a tracker for Bauldur's Gate 3 mods hosted on places like Nexus Mods or Patreon.

## Planned Features

- Captures current active mod list from local files
- Show if updates *could* be available for your mods
  - Checks Nexus API for updates (similar to MO2)
  - Checks Patrion for updates to the mod's post from the mod author.
- Lets you organize lists of mods independent of source platform.
